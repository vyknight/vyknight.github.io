This website is made using only HTML and CSS with Bootstrap. It's currently hosted on Github Pages for free.

I chose not to use any JavaScript frameworks, or JavaScript at all because I think that would be over-engineering for a simple portfolio site such as this, which has 4 pages, some text, and a picture of me.

This is the first time I've taken a website from design to deployment entirely on my own. Not the worst eh?

PokerGPT is a LLM powered poker coach, our goal is make poker coaching accessible and affordable for everyone.

Without divulging too many details, this project uses React with NextJS on the frontend, Sass and a headless component library for styling, Prisma ORM for accessing our PostgreSQL database, and a FastAPI backend.

I helped to draft the architecture, and as the main backend developer I'm responsible for running AI experiments.

Earth is an all-in-one platform for running hackathons, its goals are to address the shortcomings we find in popular hackathon mangement software such as Agorize.

Our tech stack consists of TypeScript, React, NextJS, a CSS component library, and a PostgreSQL database.

I own the judging and user directory feature, and I'm typically the first person people reach out to when they have questions regarding API and database access. Over my internship I've implemented pagination, connection pooling, and fuzzy search. I'm also the primary driver for our adoption of Prisma ORM.

Although Earth have not yet been fully released, it was widely appraised when we beta'd the program during the French multinational Capgemini's internal GenAI Hackathon. During which it was used by 1062 participants world wide.
